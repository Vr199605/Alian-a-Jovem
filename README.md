# Geração Ação

Aplicativo em Streamlit para organização e divulgação de eventos da juventude.

## Como rodar localmente:

1. Clone o repositório
2. Instale as dependências:
    ```bash
    pip install -r requirements.txt
    ```
3. Rode o app:
    ```bash
    streamlit run main.py
    ```
